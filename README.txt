# Учёт проектных задач и загрузки команды (кейс №24)
Доска задач с отчетностью и аналитикой, балансирующая загрузку исполнителя и повышающая прозрачность выполнения задач.

## Установка
Godot реализация открывается через godot 4.6 обычным путем

## Мы поддерживаем формальные API контракты в /docs/api-contracts.yaml
- Определяет все эндпоинты, схемы запросов/ответов, статусные коды.
- Является единой правдой для API проекта.
- Примеры использования, ответов и запросов можно найти в /docs/api/mocks.json
- Подробнее можно прочитать в /docs/api/api-readme.txt и wiki
